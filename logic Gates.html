Logic Gates

Binary in itself is not useful it can't do anything. To make it useful you need to use logic gates Logic gates are like slightly modified transistors that modify and change data. The most common and simple logic gates are NOT AND OR gates. The AND gates give out a 1 when they get two 1s if they get one 1 and one 0 they will output a 0. An OR gate will output a 1 as long as one of the bits it gets is a 1. It will only output a 0 if it gets 2 0s. A NOT gate will output a 0 if you input a 1 and will output a 1 if you input a 0. 0When put together these logic gates can create some of the more complex computer parts.
